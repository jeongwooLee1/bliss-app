## 1부. 고객 발송용 메시지 (복붙)

안녕하세요 원장님 🙂
블리스미(blissme.ai)로 **예약·매출·알림톡·AI 자동응대**를 한 번에 세팅해드립니다.
아래 항목을 보내주시면 저희가 다 처리합니다. **[필수]** 4가지만 먼저 주셔도 바로 시작 가능하고, **[선택]**은 나중에 추가해도 됩니다.

### ① 매장 기본 정보 [필수]
- 정식 상호(업체명) / 매장(지점) 이름
- 사업자등록증 사본
- 대표·원장 성함 / 연락처
- 매장 주소 (지점이 여러 개면 각각)
- 매장 전화번호
- 영업시간 (요일별 + 휴무일)

### ② 네이버 예약·리뷰 연동 [필수]
- 네이버 스마트플레이스에 **운영자 초대**만 해주시면 됩니다
  - 스마트플레이스 → 업체정보 → **권한 관리 → 운영자 추가** → 아이디 **`cripiss`** 초대
- 이거 하나로 예약·리뷰·예약막기·확정이 전부 자동으로 들어옵니다 (별도 프로그램 X)
- ⚠️ **[내부 필수] cripiss 예약알림 수신 토글 ON** — 운영자 초대만 하면 **예약 메일이 안 옵니다(즉시 반영 X, 30분 폴링만)**. 스마트플레이스 → **알림설정 → 예약 알림(해당 매장 선택)** → 수신자 목록에서 **`crip*** 이*우`(cripiss) 토글을 "수신중(ON)"** 으로 켜야 `housewaxing@gmail`로 메일이 와서 즉시 반영됨. (캡처: `docs/setup-captures/naver-예약알림-수신토글.png`)

### ③ 카카오 알림톡 + 문자 [필수]
- 카카오톡 채널 정보: **채널명 / 검색용 아이디 / 채널 URL**
- 카카오 채널 **비즈니스 인증** 완료
  - 카카오톡 채널 관리자센터(`center-pf.kakao.com`) → 채널관리 → **비즈니스 채널 신청** (사업자등록증, 1~2일 승인)
- **발신번호** (손님에게 찍힐 번호, 매장 대표번호 가능) + 그 번호 **통신가입증명원**
  - ※ 채널 로그인 정보를 위임해주시면 비즈니스 인증까지 저희가 대신 해드립니다

### ④ 시술 메뉴 & 가격 [필수]
- 전체 시술 메뉴 + 가격표 (**여/남 구분, 소요시간** 포함)
- 카테고리 구분 (예: 브라질리언 / 바디 / 페이스 / 케어 / 패키지)
- 패키지·정액권·회원권 상품 + 각 **혜택/적립** 내용
- 회원가(단골가) 정책 (있으면)
- **첫방문 할인·적립 등 이벤트/프로모션** 내용 (있으면 — 혜택 안내문/PDF 그대로 주셔도 됩니다)

### ⑤ 결제 [선택]
- 예약금·포인트 충전을 받으시려면: **토스페이먼츠 가맹** 후 키 (저희가 안내)
- 계좌이체 입금 자동확인: **은행 + 입금계좌번호**

### ⑥ 직원 & 근무 [선택]
- 직원 명단 (이름 / 담당 지점)
- 로그인 계정 만들 분 (원장·매니저)

### ⑦ AI 자동응대 정보 [선택 · 강력추천]
- **건물 안내**: 주차 / 엘리베이터·계단 / 대중교통 / 찾아오는 길 / 네이버·구글 지도 링크
- **자주 묻는 질문(FAQ)** + 답변 (마취크림·통증·임산부·사후관리 등)
- 스킨케어 등 부가 메뉴

### ⑧ 기타 채널 [선택]
- 인스타그램 DM / WhatsApp / 라인 자동응대 원하시면 해당 계정
- 외부 플랫폼(트레이지·크리에이트립 등) 사용 여부

**우선순위**: ①~④만 주시면 예약·매출·시술·알림톡 핵심 운영을 바로 시작합니다.
알림톡은 카카오 검수(3~5영업일) 동안 **문자(SMS)로 먼저** 나갑니다.

## 2부. 내부 세팅 매핑 (받은 것 → 우리가 하는 일)

| 받은 것 | 세팅 작업 | 비고 |
|---------|-----------|------|
| 상호/지점명/주소/영업시간 | `businesses` + `branches` 생성, business_id·branch_id 발급 | timeline_settings에 영업시간 |
| 네이버 운영자 위임 | branch `naver_biz_id` 설정 | 중앙 세션(cripiss)이 자동 폴링 — 코드 추가 0 |
| 카카오 채널 비즈니스 인증 + 발신번호 | 알리고 발신프로필 연동 → **senderKey 발급** → 템플릿 7종(rsv_confirm/1day/today/care/point/consent/chart) 모담 senderKey로 등록·검수 → `noti_config` 입력 | 알리고 계정(cripiss) 재사용. **검수는 senderKey마다 별개**(타 업체 승인분 재사용 불가, 본문은 복붙) |
| 발신번호만 | `branches.sms_callback` — SMS 즉시 발송 가능 | 알림톡 검수 전 폴백 |
| 시술 메뉴·가격 | `services` + `service_categories` 등록 (price_f/price_m/dur, is_package, pkg_count) | 네이버 자동매칭·AI가격·매출 |
| 이벤트/프로모션 | `businesses.settings.events` + `events_master_enabled` (eventEngine) | jsonb merge로 plan 보존. 시간트리거·차감형은 직원 수동 |
| 회원가 정책 | `member_price_f/m` + `member_price_rules` | |
| 토스 가맹 키 | `branches.payment_settings.tosspayments` | 예약금/포인트 |
| 은행+계좌 | `mac-daemon` 입금문자 데몬 `BLISS_KB_ACCOUNTS` 매핑 | KB·하나·수협·우리 |
| 직원 명단 | `employees_v1` + 근무표 + 멤버십 계정(accounts/app_users) | |
| 건물정보 | `branches.access_info` | AI 길안내(지어내기 차단) |
| FAQ·스킨케어 | RAG 학습문서(`documents`/`document_chunks`, gemini-embedding-2/768) | AI 자동응답 |
| 인스타/WA/라인 | 채널 토큰 등록 | AI 자동예약·응대 |

### 새 업체 추가 표준 순서
① 네이버 운영자 위임 → naver_biz_id → ② 시술상품+카테고리 → ③ 카카오 senderKey+noti_config → ④ 이벤트(settings.events) → ⑤ 결제·직원·건물·RAG(선택).
서버 멀티테넌트는 이미 업체별 스코프(`_load_ai_settings(business_id)` / `db_upsert` business_id / `review_sync sync_branch(business_id)`) — 코드 추가 없이 데이터만 넣으면 됨.
